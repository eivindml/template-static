# Static Web Template

Minimalist starting point for web project using pug (formerly Jade), sass (with .sass extension) and watch + browser-sync.

## Prerequisite
- npm
- bower

## Install

```
$ git clone https://github.com/eivindml/template-static
$ cd template-static
$ rm -rf .git
$ vim .env # Configure paths
$ npm install
$ npm install -g bower
$ bower install
```

## Usage

```
$ gulp
```
